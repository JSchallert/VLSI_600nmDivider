What's New in This Release

Version L-2016.06, June 2016

Release Notes

The HSPICE® Release Notes provide information on new features, enhancements, changes, known issues, and resolved Synopsys Technical Action Requests (STARs). To view the release notes, go to My Product Releases, locate and click HSPICE, and then click the release version of interest.

New Features and Enhancements in HSPICE L-2016.06

Enhancement to Interactive Mode

HSPICE supports multiple runs without re-parsing the input netlist, in the interactive mode. For more information, see the Multiple Runs in Interactive Mode topic in the HSPICE help.

Output template support for generic MOSFET wrapper

In addition to intrinsic MOSFETs, HSPICE now provides output template support for wrapper MOSFETs.

Support for AC Noise Output from .TRANNOISE

The results of transient noise analysis can be output in terms of a power spectral density (PSD) function. A new keyword PSD to .TRANNOISE allows you to enable PSD calculations in transient noise analysis. A new .OPTION TRNOISESEGMENTS option sets the number of window segments to use for the spectral density estimation algorithm. The PSD output allows the results from transient noise analysis to be easily compared with the results from the frequency domain .NOISE analysis.

Model Support

HSPICE L-2016.06 supports HICUM/L2 version 2.34 and BSIM-IMG version 102.7 models.

Support for MOSRA on BSIM6 Level 77 MOSFET Models

HSPICE MOSRA analysis currently supports Level 49, Level 53, Level 54, Level 57, Level 62, Level 66, Level 69, Level 70, Level 71, Level 72, Level 73, Level 76, Level 77, and external CMI MOSFET models.

Support for Reading-in Value Change Dump (VCD) Files

A new command .VCD to directly read-in VCD files.

Change in the Default Random Number Generator (RNG) for Monte Carlo Analysis

The default RNG for simple random sampling (SRS) and enhanced SRS based Monte Carlo analysis is changed to MCG. For more information, see the Random Number Generators topic in the HSPICE help.
